Host a Static Site on S3 & CloudFront

A short guide for beginners to deploy a static website using the AWS Free Tier.

Overview

This tutorial demonstrates how to host static files (HTML/CSS/JS) on Amazon S3 and serve them globally through CloudFront.

Steps

  1. Create an S3 bucket for static hosting, enable static website hosting, and set public access rules as needed.
  2. Upload your site files (e.g., `index.html`, `style.css`, `script.js`).
  3. Create a CloudFront distribution that points to your S3 bucket to get global caching and HTTPS.
  4. Optionally add a custom domain and configure DNS to point to CloudFront.
