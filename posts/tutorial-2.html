Set Up a Linux EC2 Instance

Quick start with EC2 using the AWS Free Tier.

Steps

  1. Open the EC2 console and launch a new instance (choose a Free Tier eligible t2.micro or t3.micro).
  2. Select an Amazon Linux 2 AMI, configure security group (allow SSH/HTTP), and create or use a key pair.
  3. SSH into the instance and install packages you need.
